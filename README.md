# Featured Projects

## 🔗 [shorTea](https://github.com/komallsingh/shorTea)

A production-focused URL shortener built with **Node.js, Express.js, TypeScript, and PostgreSQL**.

- Built URL shortening, redirects, analytics tracking, browser/device insights, and malicious URL detection.
- Designed with layered architecture, repository pattern, validation middleware, and structured error handling.
- Focused on scalable backend practices, API security, and efficient database design.

## 📱 [CodeBell](https://github.com/komallsingh/CodeBell)

A modern Android application built using **Kotlin, Jetpack Compose, Room Database, and REST APIs**.

- Integrates multiple competitive programming platforms for centralized contest tracking.
- Uses MVVM architecture, local caching, and efficient state management.
- Designed with clean UI principles, offline support, and scalable Android practices.
